# COBOL Program Collection

A simple collection of COBOL programs I created during my 2nd year, 1st semester.  
These programs were written as part of my coursework and exercises to learn the fundamentals of COBOL programming.

## Overview

This repository includes various COBOL programs covering basic to intermediate concepts such as:
- Data division and file handling  
- Conditional statements and loops  
- Arithmetic operations  

Each program demonstrates key COBOL features and is organized for easy reference and study.

---

## Purpose

This project serves as:
- personal archive of my learning journey in COBOL  
- reference for students or anyone starting with COBOL  
- showcase of my early programming exercises
